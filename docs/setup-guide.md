# Setup Guide

## 1. Enable Playground in Manager.io
- Open Manager.io
- Go to **Settings > Extensions**
- Click **Playground** button at bottom to enable

## 2. Get KRA Sandbox Access
- Register at https://etims.kra.go.ke
- Apply for OSCU sandbox
